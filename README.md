# TabGroup Auto

## 功能介绍

自动将新打开的标签页添加到指定的标签组中，帮助您更好地组织和管理浏览器标签页。

### 主要特性

- 自动将新标签页添加到指定标签组
- 支持忽略独立窗口（如划词翻译等）
- 简单易用的配置界面
- 完全本地运行，不收集任何用户数据

## 使用说明

1. 安装插件后，点击工具栏中的插件图标
2. 从下拉菜单中选择默认标签组
3. 选择是否忽略独立窗口
4. 点击保存即可

## 隐私政策

本插件不收集任何用户数据，所有配置信息均保存在本地浏览器中。
