# TabGroup Auto

智能标签组管理助手 — 自动将新标签页归类到指定标签组，让浏览更有条理。

---

## 功能介绍

自动将新打开的标签页加入指定标签组，并支持按 URL 规则批量归类与排序，方便管理和查找标签页。

### 主要特性

- **默认标签组**：新开标签页自动加入所选标签组
- **独立窗口识别**：可选「不将独立窗口的标签页加入标签组」（如划词翻译等小窗口）
- **URL 规则**：按 URL 匹配（如 `*github.com*`），将匹配的标签页归入指定标签组，支持「应用规则到所有标签页」与「强制移动已分组标签页」
- **标签组与排序**：在弹窗中创建新标签组、一键排序所有标签页
- **本地运行**：配置存储在浏览器本地，不收集、不上传任何用户数据

## 安装与使用

### 安装方式

1. 克隆或下载本仓库
2. 打开 Chrome 或 Edge：`chrome://extensions`（Edge 为 `edge://extensions`）
3. 开启「开发者模式」，点击「加载已解压的扩展程序」
4. 选择本项目的根目录

### 基本使用

1. 安装后点击工具栏中的扩展图标打开弹窗
2. **选择默认标签组**：在「选择默认标签组」中勾选要使用的组
3. **独立窗口**：若需排除划词翻译等独立小窗口，勾选「不将独立窗口的标签页加入标签组」
4. 点击「保存设置」生效

### 进阶配置

- **URL 规则**：在弹窗的「URL规则配置」中填写匹配串（如 `*github.com*`）、选择目标标签组，可添加多条规则，并随时「应用规则到所有标签页」
- **标签页排序**：在弹窗中点击「一键排序所有标签页」即可按当前规则整理顺序
- **选项页**：通过扩展右键菜单进入「选项」可进行更完整的配置（与弹窗中的设置一致）
